# TikTok Live Reader for NVDA

This add-on can connect to TikTok lives and extract information from them in real-time. It uses the TikTokLive Python library to access a specific live stream. The following functions are supported:

* **Saving extracted data to text files.** They are created in the `Documents\TikTok live` folder.
* **Information is saved for the following events:** comments, new followers, gifts, likes, shares, and visitors.
* **An events file is created**, in which several types of data can be recorded at once.
* **Sound playback capability** is provided for events selected in the add-on settings, such as a new comment, gift, and so on.
* **A stream statistics file is created.** It shows the current number of viewers, visitors who have passed through the live since the start of the session, total likes, number of new followers, and diamonds collected.
* **A top gifters leaderboard file is created.** They are ranked from the most active to the one with the smallest contribution, ordered by the number of diamonds sent to the host.
* **A leaderboard for users who liked the live is also available.** They are ranked on the same principle as the gifters.
