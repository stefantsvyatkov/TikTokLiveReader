TikTok Live Reader

Add-on ini terhubung ke siaran TikTok Live dan mengekstrak informasi secara real-time. Add-on menggunakan library Python TikTokLive untuk mengakses siaran live tertentu. Fitur yang didukung:

  • Mencatat data hasil ekstraksi ke file teks di folder Documents\TikTok live.
  • Melacak peristiwa: komentar, pengikut baru, hadiah, likes, share, dan pengunjung.
  • File events yang dapat merekam banyak jenis data secara bersamaan.
  • Notifikasi suara untuk peristiwa yang dipilih, misalnya komentar baru, hadiah, dan lainnya.
  • Statistik siaran: menampilkan jumlah penonton saat ini, total pengunjung sesi, total likes, pengikut baru, dan diamond yang terkumpul.
  • Peringkat Top Gifters: mengurutkan pengguna dari paling aktif ke paling sedikit berdasarkan diamond yang dikirim.
  • Peringkat Top Likers: mengurutkan pengguna berdasarkan total likes selama siaran.

TikTok Live Reader membantu host atau penonton memantau data penting siaran tanpa alat eksternal. Data disimpan di file teks dalam Documents\TikTok live dan dapat ditinjau kembali nanti. Add-on mengekstrak tipe data berikut ke file terpisah:

  • comments.txt - Mencatat semua komentar yang terdeteksi selama live beserta timestamp.
  • followers.txt - Mencatat semua pengguna baru yang mengikuti kreator selama sesi berjalan.
  • gifts.txt - Mencatat semua hadiah yang diterima: pengguna, jenis hadiah, jumlah, dan timestamp.
  • likes.txt - Menampilkan siapa yang memberi like, kapan, dan berapa kali.
  • shares.txt - Mencatat pengguna yang membagikan siaran live beserta timestamp.
  • stats.txt - Menampilkan penonton saat ini, pengikut baru, total likes, total pengunjung sesi, dan total diamond.
  • top gifts.txt - Peringkat pengguna yang mengirim hadiah, dari paling aktif sampai paling sedikit.
  • top likes.txt - Peringkat pengguna berdasarkan total likes.
  • visitors.txt - Mencatat semua pengunjung yang masuk selama sesi berjalan beserta timestamp.
  • events.txt - Log gabungan berurutan untuk berbagai peristiwa (komentar, pengikut, hadiah, dan lainnya), dapat ditentukan oleh pengguna.

Pengaturan

TikTok Live Reader memiliki jendela pengaturan sendiri. Buka dengan pintasan NVDA+Shift+Control+L. Jendela ini memiliki 3 tab:

General

Bagian ini memuat pengaturan berikut:

  • Username - Masukkan username host TikTok live tanpa simbol @.
  • Clear text files on startup - Menghapus isi file teks saat add-on dimulai.
  • Clean usernames - Menghapus emoji dan simbol Unicode dari username agar lebih kompatibel dengan screen reader. Saat aktif, username hanya berisi huruf, angka, dan tanda baca.
  • Connection retry count - Menentukan berapa kali add-on mencoba tersambung ke siaran live.

Events

Kotak centang ini menentukan jenis data apa yang dicatat di file Events. Pilihan mencakup Comments, Followers, Gifts, Likes, Shares, dan Visitors. Opsi tambahan:

  • Play sounds for the selected events - Memutar suara singkat khusus saat peristiwa terdaftar terjadi.
  • Audio volume - Mengatur level suara notifikasi.
  • Learn sounds - Memutar deskripsi berbagai suara agar pengguna dapat mengenalinya.

Auto speak

Bagian ini menentukan peristiwa apa yang dibacakan ketika fungsi Auto speak aktif. Aktif/nonaktifkan dengan NVDA+Shift+Control+S. Saat peristiwa terpilih terjadi, peristiwa dibacakan otomatis (misalnya "New follower: Username" atau "Username shared the live").

Pengumuman dan suara diputar dengan jeda satu detik. Keduanya disimpan dalam file sementara agar pelaporan tetap berurutan saat traffic live tinggi. Notifikasi suara dan Auto speak bekerja secara independen.

Cara menggunakan add-on

Setelah konfigurasi selesai, tekan NVDA+Shift+Control+T untuk terhubung. Status koneksi akan diumumkan otomatis. Hotkey berikut hanya aktif selama sesi berjalan:

  • Control+Shift+Right Arrow - File berikutnya (misalnya Comments, Events, Followers, Gifts, dll).
  • Control+Shift+Left Arrow - File sebelumnya.
  • Control+Shift+Down Arrow - Item berikutnya di file saat ini (misalnya komentar, like, share).
  • Control+Shift+Up Arrow - Item sebelumnya di file saat ini.
  • Control+Shift+Home - Pindah ke item pertama di file saat ini.
  • Control+Shift+End - Pindah ke item terakhir di file saat ini.
  • NVDA+Shift+Control+V - Melaporkan jumlah penonton saat ini.
  • NVDA+Shift+Control+S - Mengaktifkan atau menonaktifkan Auto speak.
  • NVDA+Shift+Control+R - Mengosongkan isi file teks secara manual.

Tekan NVDA+Shift+Control+T lagi untuk mengakhiri sesi saat ini. Data tetap tersedia di folder Documents\TikTok live.

Catatan

  • Membuka pengaturan akan menghentikan koneksi aktif agar perubahan dapat diterapkan dengan stabil.
  • Jika "Clear files on startup" nonaktif, data baru akan ditambahkan ke log lama. Gunakan NVDA+Shift+Control+R untuk mengosongkannya secara manual.
  • Add-on mengingat posisi terakhir Anda di file jika file tidak dibersihkan otomatis.

Batasan

  • Add-on tidak dapat menampilkan daftar semua penonton saat ini karena library TikTokLive tidak melaporkan saat pengguna keluar dari siaran.
  • Peringkat berbasis durasi tonton tidak dapat dibuat karena tidak ada event saat pengguna meninggalkan siaran.
  • Saat proses penerjemahan add-on ke bahasa lain, ditemukan bahwa nama hadiah TikTok sangat banyak dan sering berubah. Menjaga terjemahan manual untuk semuanya tidak praktis, jadi nama hadiah dilaporkan dalam bahasa Inggris.
