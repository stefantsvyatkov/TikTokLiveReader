TikTok Live Reader

This add-on connects to TikTok Live streams and extracts information in real-time. It uses the TikTokLive Python library to access a specific live stream. The following features are supported:
  • Logging extracted data into text files created in the Documents\TikTok live folder.
  • Tracking events: comments, new followers, gifts, likes, shares, and visitors.
  • An events file that can record multiple types of data simultaneously.
  • Audio notifications for selected events, such as new comments, gifts, etc.
  • Stream statistics: displays current viewers, total session visitors, total likes, new followers, and diamonds collected.
  • Top Gifters leaderboard: ranks users from most active to least active based on diamonds sent.
  • Top Likers leaderboard: ranks users based on their total likes during the stream.
TikTok Live Reader allows hosts or viewers to monitor essential stream data without external tools. Data is saved in text files in Documents\Tiktok live and can be reviewed later. The add-on extracts the following data types into separate files:
  • comments.txt - Records all comments detected during the live stream with timestamps.
  • followers.txt - Registers all new users who followed the live creator during the current session.
  • gifts.txt - Logs all received gifts, reporting the user, gift type, quantity, and timestamp.
  • likes.txt - Shows who liked the stream, when, and how many times.
  • shares.txt - Records the users who shared the live stream along with the timestamp.
  • stats.txt - Shows current viewers, new followers, total likes, total session visitors, and total diamonds received.
  • top gifts.txt - A leaderboard of users who sent gifts, ranked from most to least active.
  • top likes.txt - A leaderboard of users ranked by their total likes.
  • visitors.txt - Records all visitors who entered the stream during the current session with timestamps.
  • events.txt - A combined chronological log for various events (comments, followers, gifts, etc.). They can be defined by the user.

Settings

TikTok Live Reader has its own settings window. It can be opened with keyboard shortcut NVDA+Shift+Control+L. The settings window contains 3 tabs:

General

This section includes the following settings:
  • Username - Enter TikTok live host's username without @ symbol.
  • Clear text files on startup - Removes the content of text files when the add-on starts.
  • Clean usernames - Removes emojis and Unicode symbols from usernames. This ensures better compatibility with screen readers. When the option is enabled, the usernames contain only letters, numbers and punctuation.
  • Connection retry count - Defines how many times the add-on should try to connect to the live stream.

Events

These check boxes determine which data types are recorded in the Events file. Options include Comments, Followers, Gifts, Likes, Shares and Visitors. Additional options:
  • Play sounds for the selected events - Plays a specific short sound when a registered event occurs.
  • Audio volume - Adjusts the notification sound level.
  • Learn sounds - Plays a description of the different sounds so the user can familiarize themselves with them.

Auto speak

This section defines which events are announced when the Auto speak function is enabled. Activate it via keyboard shortcut NVDA+Shift+Control+S. When a selected event occurs, it is read automatically (e.g., "New follower: Username" or "Username shared the live"). Announcements and sounds are played with a one second pause between them. They are stored in a temporary files to ensure they are reported in order during live streams with high traffic. Audio notifications and Auto speak functions work independently.

How to use the add-on

Once configured, press NVDA+Shift+Control+T to connect. Connection status is announced automatically. The following hotkeys work only during an active session:
  • Control+Shift+Right Arrow - Next file (e.g., Comments, Events, Followers, Gifts, etc.).
  • Control+Shift+Left Arrow - Previous file.
  • Control+Shift+Down Arrow - Next item in the current file (e.g., comment, like, share).
  • Control+Shift+Up Arrow - Previous item in the current file.
  • Control+Shift+Home - Moves focus to the first item in the current file.
  • Control+Shift+End - Moves focus to the last item in the current file.
  • NVDA+Shift+Control+V - Reports the current viewer count.
  • NVDA+Shift+Control+S - Toggles the auto speak function on or off.
  • NVDA+Shift+Control+R - Manually clears the content of text files.
Press NVDA+Shift+Control+T to terminate the current session. Data remains available in the Documents\TikTok live folder.

Notes

  • Opening settings terminates the active connection to ensure stability when applying changes.
  • If "Clear files on startup" is disabled, new data is appended to the existing logs. Use NVDA+Shift+Control+R to clear them manually.
  • The add-on remembers your last position in the files if they are not cleared automatically.

Limitations

  • The add-on cannot show a list of all current viewers because the TikTokLive library does not report when a user leaves the stream.
  • A leaderboard based on watch time cannot be created due to the lack of user leave events.
  • During the translation of the add-on in Bulgarian, which is my native language, it was determined that TikTok gifts are too numerous and frequently updated by the social network. Maintaining a manual translation for all of them is not viable. therefore, gift names will be reported in English.
